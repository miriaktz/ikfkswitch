IK/FK SWITCH

1. select the 3 joints you want to create the switch for PLUS one joint above (e.g.: l_clav_jnt, l_armUp_jnt, l_elbow_jnt, l_wrist_jnt)
2. click 'Create IK Setup'
3. select the 4 joints again
4. click 'Create FK Setup

5. select all 3 original joints, then all 3 ik joints, then all 3 fk joints
6. click 'Create Connections'

7. select switch_ctrl, then zero group of pv_ctrl + wrist_ik_ctrl, then zero group of the first fk_ctrl
8. click 'Create SDK Setup'
